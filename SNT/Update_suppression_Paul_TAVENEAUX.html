Comment fonctionne l'update suppression ?

Minecarft est un jeu codé en java. Dans le jeu toutes les updates ("les mise à jour"), qui sont toutes les actions géré par le serveur, sont stocké dans une pile. En java, les piles ont une valeur maximum. Toutes les actions qui sont présentes dans cette pile seront effectués, mais si il n'y a plus de place dans cette pile, tout action qui ne rentre pas dans cette pile ne seront pas effectués. Donc si on trouve un moyen de créer en 1 tick suffisament d'actions pour provoquer le remplisage de cette pile, nous seront donc capable d'enpècher une mise à jour de se propager vers certain blocs
